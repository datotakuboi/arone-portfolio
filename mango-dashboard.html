🥭 Carabao Mango Sensor Dashboard

IoT-based environmental monitoring system for carabao mango cultivation with real-time data visualization and predictive analytics

Dashboard Overview

Real-time monitoring of mango ripeness status with key metrics including chlorophyll levels, carotenoid content, and NIR values. The dashboard displays current scan status with timestamp updates.

Features:

Real-time Status Key Metrics Data Visualization IoT Integration

Detailed Metrics & History

Complete sensor data including chlorophyll, carotenoid, and NIR readings. The dashboard tracks sweetness levels and maintains a history of recent scans with accurate timestamps for quality control.

Capabilities:

Sensor Data Analytics Historical Tracking Quality Assessment

Project Details

📊 Sensors

Environmental monitoring sensors for temperature, humidity, light intensity, and spectral analysis

📈 Metrics

Chlorophyll content, carotenoid levels, NIR readings, and sweetness assessment

🔄 Real-time Updates

Live dashboard updates with scan history and predictive ripeness indicators

Tech Stack & Tools

IoT Sensors Data Collection Real-time Analytics Dashboard UI Database Spectral Analysis
