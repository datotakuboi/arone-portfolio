## 🚀 Getting Started

### Prerequisites
- A modern web browser
- Python (optional, for local server)
- Git

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/yourusername/portfolio.git
cd portfolio
```

2. Serve locally using Python:
```bash
python -m http.server 8000
```

3. Open your browser and navigate to:
```
http://localhost:8000
```

## 🌓 Dark Mode

The portfolio includes built-in light and dark mode toggle. Your preference is saved automatically using LocalStorage and will persist across sessions.

Click the moon/sun icon in the navigation bar to switch themes.

## 📸 Profile Photo

Add your profile photo to display in the hero section:

1. Save your professional headshot as `profile.jpg`
2. Place it in the `images/` folder
3. The image should be 500x600px or larger for best results
